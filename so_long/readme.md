##Prerequisite
Only working on Linux Kernel version 5.15 or Upper and GUI with X-Session
Stable at Ubuntu 22.04

##How To Prepare
1. Instruct "sudo apt-get update && sudo apt-get install xorg libxext-dev zlib1g-dev libbsd-dev"
2. Instruct "make"

##How To run
1. Instruct './so_long "An ending with '.ber' file"' //Some test files at ./maps
2. Enjoy the 'so_long' Game!
